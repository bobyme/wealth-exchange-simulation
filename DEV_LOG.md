# 財富分配模擬器：開發與討論日誌 (Development Log)

## 📅 紀錄日期：2026-03-07

### 1. 核心需求與目標 (Core Requirements)
*   **全球擴張**：將原本的三國對比（美、台、北歐）擴展為四國實證模擬（美、台、日、中）。
*   **機制深化**：引入「年化 GDP 成長」與「勞動貢獻率 (Labor Share)」指標，模擬生產力進步對貧富差距的衝擊。
*   **工程標準化**：建立標準的 Git 工作流，清理暫存檔案，並完成 GitHub 雲端同步。

### 2. 技術更新紀錄 (Technical Updates)
*   **`wealth_exchange.py` 引擎升級**：
    *   新增 `labor_growth_step` 參數，模擬生產力隨時間的複利增長。
    *   優化數值穩定性：將溢出保護閾值提升至 `1e15`。
    *   改進 UBI (全民均分) 邏輯，確保稅收再分配的公平性。
*   **新增分析腳本**：
    *   建立 `run_country_comparison.py`，封裝了四種截然不同的經濟參數組。
*   **報告優化**：
    *   更新 `ECON_MODELS.md`，納入日本的「低增長公平」與中國的「高增長兩極分化」分析。

### 3. 核心洞察 (Key Insights)
*   **資本稅是公平的唯一煞車**：日本與美國的對比證明，沒有足夠的資產稅（Japan 25% vs USA 5%），基尼係數終將失控。
*   **勞動貢獻率的警訊**：在高成長模式下，若資本回報 (8-10%) 遠高於 GDP 成長 (3-5%)，勞動者努力工作的成果會被資本迅速稀釋。
*   **儲蓄斜率與階級固化**：美國的極端儲蓄分佈（富人 98% 儲蓄 vs 窮人 2% 儲蓄）是社會斷層的主因。
