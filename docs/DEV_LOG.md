### 主題：Schelling 居住隔離模型擴展與仕紳化機制

#### 1. 模型視覺化解讀

本次討論釐清了 `human_simulation.py` 輸出圖的解讀方式：

- **左側居住分布圖**：深灰=空屋、青綠=Group1（窮人）、金黃=Group2（富人）
- **右側地價熱圖**（`price_enabled=True` 時出現）：黑→紅→白對應低→高地價
- **標題數字**：Step / Unhappy（不滿意人數）/ Moved（實際搬家人數）
- **紅色虛線圓圈**：CBD 市中心容忍加成範圍，圈內兩族均感到「滿意」，自然形成雜居區

修正中文字體顯示問題，在 `human_simulation.py` 頂部加入：
```python
plt.rcParams['font.family'] = ['Arial Unicode MS', 'PingFang SC', 'Heiti TC', 'sans-serif']
plt.rcParams['axes.unicode_minus'] = False
```

#### 2. 仕紳化機制（`price_enabled=True`）

新增參數與說明：

| 參數 | 預設值 | 作用 |
|------|--------|------|
| `price_enabled` | False | 啟用地價系統 |
| `price_floor` | 0.5 | 地價下限 |
| `price_ceiling` | 3.0 | 地價上限 |
| `price_cbd_premium` | 1.0 | 市中心初始溢價 |
| `price_appreciation_rate` | 0.05 | 富人入住每步漲幅 |
| `price_decay_rate` | 0.02 | 空置/窮人每步跌幅 |
| `income_limit_g1` | 1.5 | 窮人負擔上限，超過強制搬離 |
| `cbd_gravity_g1` | 0.2 | 窮人市中心偏好（低） |
| `cbd_gravity_g2` | 0.95 | 富人市中心偏好（高） |

**為何市中心仍有窮人的原因分析：**
- 初始中心地價 = `price_floor + price_cbd_premium = 1.5`，剛好等於窮人上限，需累積漲幅才觸發
- `cbd_gravity_g1` 只影響「搬去哪」，不影響「現在住哪」
- CBD 容忍加成讓中心窮人永遠「滿意」，不會主動離開

#### 3. 模型解釋力與限制討論

**有解釋力的面向：**
- Schelling 核心命題：溫和個人偏好 → 極端整體隔離
- 市中心雜居、外圍隔離的同心圓結構
- 仕紳化的空間蔓延邏輯（富人入住 → 地價外溢 → 窮人被推出）
- 購買力不對稱的資源競爭結構

**主要限制：**
- 搬家決策過於簡化（假設全局資訊、沒有社會網絡）
- 地價機制為線性，缺乏預期心理與政策干預
- 人口固定，無出生/移入/移出
- 空間均質，無地理限制（河流、捷運等）
- 無法區分「偏好造成」vs「歷史歧視造成」的隔離

#### 4. 三國模擬擴展（`simulation_report.py`）

新增日本場景，三國參數設計依據：

| 參數 | 美國 | 台灣 | 日本 |
|------|------|------|------|
| 隔離軸 | 種族 | 經濟階層 | 職業/階層 |
| `friction_cost` | 0.15（低） | 0.50（中高） | 0.65（極高） |
| `cbd_gravity` | 0.20（郊區化） | 0.85（強） | 0.90（一極集中） |
| `cbd_gravity_g2` | 0.20 | 0.95 | 0.95 |
| `cbd_gravity_g1` | 0.20 | 0.20 | 0.15 |

**模擬結果：**

| 國家 | 收斂步數 | 最終隔離指數 | 特徵 |
|------|---------|------------|------|
| 美國 | 71 步 | 0.982 | 快速收斂，隔離最徹底 |
| 台灣 | 500 步 | 0.919 | 緩慢演化，階層分化 |
| 日本 | 492 步 | 0.951 | 遲緩但固化，初始就偏高 |

**三國驅動機制差異：**
- 美國：偏好導向（主動選擇同族鄰居）
- 台灣：價格導向（高所得搶市中心，低所得被動外移）
- 日本：慣性主導（極低流動性使現有格局固化）

**政策意涵：** 針對機制對症下藥——提高美國搬家成本（租金穩定）、台灣補貼窮人進入市中心、日本促進居住流動——才能從根本緩解隔離。
